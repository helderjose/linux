----------------------------------

remover programas da inicialização.
Para isso, habilite a apresentação dos ítems ocultos no menu executando no terminal o comando:
sudo sed -i "s/NoDisplay=true/NoDisplay=false/g" /etc/xdg/autostart/*.desktop
Vá em Aplicativos se Sessão no menu do Ubuntu e remova as entradas desnecessárias para a sua configuração. 
-------------------------------------

trocar de ambiente gráfico
encerre a sessão, clique no logo ubuntu para escolher.
-----------------------------------

sudo apt-get install gnome-session-fallback
