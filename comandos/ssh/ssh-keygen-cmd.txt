Cenário: Criar um script .sh que conecte via ssh sem pedir senha.

+++++ Gerar as chaves ssh.
no laptop execute:
cd ~/.ssh
ssh-keygen -t rsa
Basta seguir as instruções e pressionar "Enter" para as opções padrão.

cd ~/.ssh
