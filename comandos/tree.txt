Exibe a árvore de diretório.
Mostra a quantidade de arquivos e diretórios no final.


man tree
tree --help

tree -d -L 1 /home


Opções:
-d  list directories only
-L  qlevel, max display depth of the directory tree



--------------------
----- exemplos ----
--------------------
tree -L 2 pasta-alvo

----- salvar em um arquivo -----
tree -L 2 pasta-alvo > arquivo-saida.txt

----- ignorar uma pasta -----
tree -L 2 -I "pasta-para-ignorar-1|ignorar-2" pasta-alvo

ver a quantidade de diretorios,
